# Arduino-ECG

Some arduino code written for the AD8232 heart rate sensor.

# Pins

GND    - GND

3.3V   - 3.3V

OUTPUT - A0

LO-    - Digital Pin 10

LO+    - Digital Pin 11

# Electrode Placement
If you're copying this, you should use a continuity tester to check, don't rely on this.

RED - STERNUM                  
YELLOW - LEFT ARM              
GREEN - 5TH RIB DOWN, LEFT SIDE
